# jBrow
基于nwjs的轻量级浏览器

## 特征
* 轻
* 简
* 无痕 <sub style="font-size:6pt">（只要你愿意）</sub>
* 可拓展

## 安装
1. 下载 [nwjs](http://nwjs.io/) 接着解压它
2. 把 [jBrow](https://github.com/TitanSnow/jBrow/releases) (package.nw) 放入nwjs的文件夹
3. 运行可执行文件"nw"，then enjoy yourself !
